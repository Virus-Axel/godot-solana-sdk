Detailed Description

Manager for RPC HTTP API communication.

Interacts with an RPC node on a Solana cluster. This class takes user requests and sends them sequentially or in parallel to an RPC endpoints. The SolanaClient also manages the web socket interface.

◆ set_timeout()

void godot::SolanaClient::set_timeout ( float timeout)

Set the timeout of requests.

Sets timeout to use when calling RPC endpoint. When a request takes longer that the timeout, the SolanaClient will stop waiting and throw a timeout response instead. This will trigger response signal too, but with a timeout error.

Parameters
timeoutTimeout in seconds.

◆ set_url_override()

void godot::SolanaClient::set_url_override ( const String & url)

Set the url to use by this client instance.

The SolanaClient uses the URL in project settings. However, this method overrides that URL. By calling this method this instance will use the provided URL instead. Once the URL override is set it cannot be unset.
