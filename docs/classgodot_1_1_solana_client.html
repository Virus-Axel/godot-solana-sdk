Detailed Description

Manager for RPC HTTP API communication.

Interacts with an RPC node on a Solana cluster. This class takes user requests and sends them sequentially or in parallel to an RPC endpoints. The SolanaClient also manages the web socket interface.

◆ send_transaction()

void godot::SolanaClient::send_transaction ( const String & encoded_transaction,
uint64_t max_retries = UINT64_MAX,
bool skip_preflight = false )

Submit a signed transaction to the cluster for processing.

If the node's rpc service receives the transaction, this method immediately succeeds, without waiting for any confirmations. A successful response from this method does not guarantee the transaction is processed or confirmed by the cluster.

Parameters
encoded_transactionSigned and encoded serialized transaction.
max_retriesMaximum number of times for the RPC node to retry sending the transaction to the leader. If this parameter not provided, the RPC node will retry the transaction until it is finalized or until the blockhash expires.
