Detailed Description

A helper class for transaction messages.

This class handles the transaction message operations. The class will manage the message state and is responsible of creating a serialization of message bytes.

void Message::create ( const PackedByteArray & bytes)

Recreates a complete transaction message.

This method parses the message header and performs some error checks. If valid the method reads all the properties of the serialized message. This method can be used to recreate a message from a serialized transaction.
