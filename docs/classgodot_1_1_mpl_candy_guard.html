Create instruction to mint from a candy machine wrapped by a candy guard.

Can mint both NFT and pNft. Only when the transaction is succesfully validated by the MplCandyGuard program, it is forwarded to the Candy Machine.

Note
In Metaplex program this instruction is referred to as mintV2.
Parameters

guard_account_idCandy guard key. Can be retrieved using new_associated_candy_guard_key.

collection_mintMint account of the collection.
